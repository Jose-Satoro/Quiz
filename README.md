# Quiz sobre HTML, CSS e JavaScript

Este projeto contém um quiz simples com 5 perguntas sobre HTML, CSS e JavaScript.

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Responda todas as perguntas.
3. Clique em "Enviar respostas".

O aplicativo vai mostrar:
- Quantas perguntas você acertou.
- Uma mensagem de incentivo ou sugestão para praticar.
- Quais questões foram corretas e quais foram erradas.
